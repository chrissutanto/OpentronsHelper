Connection Guide

Connect to the OT2

First, make sure that the OT-2 is powered on and connected to the power supply and computer. The blue light should be solid.

Open the Opentrons app and select the "robot" tab in the top left.

Refresh the list until the OT-2's name shows up. Toggle the switch next to the name to establish a connection.

Now, you should be able to run protocol scripts, install pipettes, and upload custom labware definitions.
